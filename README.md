# Job Description Skill Extractor

## Problem Statement

Job descriptions contain important hiring requirements hidden inside long paragraphs.

Recruiters often spend significant time manually extracting required skills, experience, and education.

This project uses Generative AI to automatically extract structured information from job descriptions.

## Input

Job Description Text

## Output

{
    "skills": [],
    "experience": "",
    "education": ""
}

## Objective

Extract only the information explicitly mentioned in the job description without making assumptions.
